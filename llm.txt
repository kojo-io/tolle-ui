# Tolle UI - Angular Component Library

Tolle UI is a modern UI component library for Angular built with Tailwind CSS and the Angular CDK. It emphasizes accessibility, customizability, and a premium aesthetic similar to shadcn/ui.

## Installation & Getting Started

### 1. Install Dependencies
```bash
npm install @tolle_/tolle-ui date-fns remixicon @melloware/coloris embla-carousel clsx tailwind-merge class-variance-authority @floating-ui/dom
```

### 2. Configure Tailwind CSS
In your `tailwind.config.js`:
```javascript
const { tolleUi } = require('@tolle_/tolle-ui/tailwind');

module.exports = {
  darkMode: 'class',
  presets: [
    require('@tolle_/tolle-ui/preset') // Point to your library preset
  ],
  content: [
    "./src/**/*.{html,ts}",
    "./node_modules/@tolle_/tolle-ui/**/*.{html,ts,mjs,html}"
  ],
  plugins: [],
};
```

### 3. Setup Global Styles
In your `angular.json`:
```json
{
  "styles": [
    "src/styles.css",
    "node_modules/@tolle_/tolle-ui/theme.css"
  ]
}
```

### 4. Provide Configuration
In your `app.config.ts`:
```typescript
import { ApplicationConfig } from '@angular/core';
import { provideTolleConfig } from '@tolle_/tolle-ui';

export const appConfig: ApplicationConfig = {
  providers: [
    provideTolleConfig({
      primaryColor: '#8b5cf6',
      radius: '0.5rem',
      darkByDefault: false
    })
  ]
};
```

## Theming

Tolle UI uses CSS variables for theming, which are managed by `ThemeService` and `provideTolleConfig`.

### Config Options:
- `primaryColor`: Hex code for the brand color.
- `radius`: CSS border-radius value (e.g., '0.5rem').
- `darkByDefault`: Boolean to start in dark mode.

### Overriding at Runtime:
Use `ThemeService.updateConfig(partialConfig)` to change colors or radius dynamically.

## Components Reference

### Button
Selector: `tolle-button`
Import: `import { ButtonComponent } from '@tolle_/tolle-ui';`
Usage: `<button tolle-button variant="outline" size="sm">Click</button>`
Inputs:
- `variant`: 'default' | 'destructive' | 'outline' | 'secondary' | 'ghost' | 'link'
- `size`: 'default' | 'xs' | 'sm' | 'lg' | 'icon' | 'icon-xs' | 'icon-sm' | 'icon-lg'
- `busy`: boolean
- `disabled`: boolean

### Input
Selector: `tolle-input`
Import: `import { InputComponent } from '@tolle_/tolle-ui';`
Usage: `<tolle-input label="Name" [(ngModel)]="name"></tolle-input>`
Inputs:
- `label`, `hint`, `errorMessage`: string
- `type`: 'text' | 'password' | 'email' | etc.
- `size`: 'xs' | 'sm' | 'default' | 'lg'
- `error`: boolean
- `disabled`: boolean

### Textarea
Selector: `tolle-textarea`
Import: `import { TextareaComponent } from '@tolle_/tolle-ui';`
Inputs: `label`, `rows`, `autoGrow`, `maxLength`, `showCharacterCount`, `error`, `disabled`.

### Slider
Selector: `tolle-slider`
Import: `import { SliderComponent } from '@tolle_/tolle-ui';`
Usage: `<tolle-slider [(ngModel)]="val" [min]="0" [max]="100"></tolle-slider>`
Range: `[(ngModel)]="[min, max]"`
Inputs: `min`, `max`, `step`, `disabled`.

### Select
Selector: `tolle-select`, `tolle-select-item`
Import: `import { SelectComponent, SelectItemComponent } from '@tolle_/tolle-ui';`
Usage: `<tolle-select><tolle-select-item value="1">One</tolle-select-item></tolle-select>`
Inputs: `placeholder`, `searchable`, `size`, `disabled`.

### Switch
Selector: `tolle-switch`
Import: `import { SwitchComponent } from '@tolle_/tolle-ui';`
Usage: `<tolle-switch [(ngModel)]="enabled"></tolle-switch>`
Inputs: `size`, `disabled`.

### Checkbox
Selector: `tolle-checkbox`
Import: `import { CheckboxComponent } from '@tolle_/tolle-ui';`

### Sheet (Off-canvas)

Inputs: `side` ('top'|'bottom'|'left'|'right'), `rounded` (boolean).

### Modal / Alert Dialog

Import: `import { ModalService, AlertDialogService } from '@tolle_/tolle-ui';`
Usage: `this.modalService.open({ title: 'Title', content: MyComponent })`.
