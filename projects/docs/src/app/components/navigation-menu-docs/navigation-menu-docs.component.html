Installation

Add it with the CLI (you own the source), or import the components if you use the package.

Examples

Site navigation

A trigger item opens a panel of links; each link projects a [label] and an optional [description] line. Give the panel a fixed width so the grid inside it has something to lay out against.

Flat and dropdown entries

An item does not need a trigger — drop a link straight into it for a flat entry, and mark the current page with [active]="true". Here align="start" pins the whole row to the left.

Hover to switch

With two triggers side by side, moving the pointer from one to the other swaps panels without closing in between. closeDelay is the grace period that lets the pointer travel diagonally into an open panel; openChange reports which item is open. Escape closes, and so does a click outside.
