Add it with the CLI (you own the source), or import the components if you use the package.

Examples

Variants

Four kinds of row, each with its own emphasis and placement: separator draws a labelled rule, default centres a quiet system note, status hugs the transcript's leading edge for live activity, and error reports a failure.

Grouping tool activity

A run of tool calls is one step, not five interruptions. Wrap them in tolle-marker-group so they read as a single block the reader can skim past.

In a transcript

Markers sit between messages as siblings, not inside them — they belong to the transcript, not to any one sender.
