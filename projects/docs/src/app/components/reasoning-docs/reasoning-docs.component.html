Installation

Add it with the CLI (you own the source), or import the components if you use the package.

Anatomy

Compose the trace from a trigger and its content. The trace is collapsed by default and owns the shared state, so the trigger and content need no bindings of their own.

Examples

Basic

A finished trace. The trigger reports the thinking time from duration; the content only renders while expanded, and preserves the newlines in the trace.

Streaming a trace

Hold streaming true while the chunks arrive: the trigger reads “Thinking…” with a pulsing icon. Drop it and set duration when the trace lands, and the same label becomes “Thought for N seconds”.

Custom labels and rich content

Relabel the trigger for a non-chat flow, and project markup into the content when the trace is more than a paragraph. openChange reports user toggles only — mounting the trace never emits.
