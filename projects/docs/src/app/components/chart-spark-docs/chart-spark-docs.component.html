Installation

Add it with the CLI (you own the source), or import the component if you use the package. It composes tolle-chart internally, so that registry dependency comes along with it.

Anatomy

A thin wrapper, not a second chart engine: it draws one series into a tolle-chart with the grid, axes, legend and hover layer left off. It inherits --chart-1 the same way any other single-series chart in this library does.
