Installation

Add it with the CLI (you own the source), or import the component if you use the package.

The option type

Every entry you pass to [options] is a ComboboxOption. Only label and value are required — keywords add synonyms that match while searching without showing up in the list.

The combobox implements ControlValueAccessor, so it drops straight into ngModel or reactive forms and writes the option's value to the control. Reach for tolle-select instead when the list is short and needs no search, and tolle-multi-select when more than one value can be chosen.

Examples

Basic with ngModel

Bind the selected value with [(ngModel)]. Options marked disabled stay visible but cannot be picked.

Reactive forms

Use formControlName like any other control. Calling control.disable() reaches the combobox through setDisabledState() and greys out the trigger — note that a disabled control also drops out of form.value.

Sizes, invalid state and keywords

Four trigger heights, plus [invalid]="true" for the destructive border and focus ring. These options carry keywords, so typing “wip” finds In progress and “shipped” finds Done.
