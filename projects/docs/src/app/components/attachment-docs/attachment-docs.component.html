Installation

Add it with the CLI (you own the source), or import the components if you use the package.

Examples

Basic

Pass the raw byte count to size — the card formats it. The icon is picked from the MIME type, and an image/* type with a url renders a thumbnail instead.

Upload states

An upload that fails silently is worse than one that never started. Drive state and progress from your upload task, and give the failure case an errorLabel that says what went wrong rather than just that something did.

Attached to a message

tolle-attachment-group wraps several cards into a row inside tolle-message-content. Anything in tolle-attachment-actions appears on hover or focus and swallows its own clicks, so the card's open does not fire too.
