Breadcrumb Component Style Reference

Component Area Selector What It Targets
Breadcrumb Host tolle-breadcrumb The main breadcrumb component host element
Navigation Container tolle-breadcrumb nav The navigation element with breadcrumb role and base styles
Breadcrumb List tolle-breadcrumb ol Ordered list containing all breadcrumb items and separators
Breadcrumb Item Host tolle-breadcrumb-item Individual breadcrumb item component host
Item Container tolle-breadcrumb-item li List item container for each breadcrumb element
Breadcrumb Link Host tolle-breadcrumb-link Breadcrumb link component host with active/inactive states
Active Link tolle-breadcrumb-link span[aria-current="page"] Current/active page breadcrumb link (non-clickable)
Inactive Link tolle-breadcrumb-link div Clickable/inactive breadcrumb link wrapper
Breadcrumb Separator Host tolle-breadcrumb-separator Separator component host between breadcrumb items
Separator Item tolle-breadcrumb-separator li Separator list item containing the separator icon
Default Separator Icon tolle-breadcrumb-separator i.ri-arrow-right-s-line Default arrow icon used as breadcrumb separator
