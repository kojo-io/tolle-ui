Installation

Add it with the CLI (you own the source), or import the components if you use the package.

Anatomy

Fill a row with pills. The row relays whichever pill was picked, so you can bind selected once instead of on every pill — but each pill also has its own selected output, and works perfectly well on its own.

Examples

Basic

The projected content is the label and value is what gets emitted, so the two can differ.

Variants, sizes and icons

Three visual styles and three sizes. Add a remixicon class to lead the label, and widen the row with gap.

Overflowing rows

Pills never wrap — the row scrolls sideways with a faded edge, so a long list of follow-ups stays one line tall under a message.
