Theming in Tolle UI

Tolle UI is built with theme-first principles. Instead of hard-coding colors into components, everything is driven by semantic tokens, CSS variables, and Tailwind utilities, giving you full control over branding and appearance.

Design Philosophy

Tolle UI uses foreground and background tokens instead of direct colors.

Example

  • bg-background
  • text-background

This abstraction ensures:

  • Consistent theming across components
  • Easy dark mode support
  • Seamless brand customization

You never theme components directly first — you theme the system, and components inherit automatically.

Tolle UI is not about chasing trends. It’s about making day-to-day Angular development calmer, clearer, and more predictable.
