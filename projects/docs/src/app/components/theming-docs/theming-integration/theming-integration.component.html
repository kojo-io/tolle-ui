Adding the Tolle UI Theme File

Tolle UI ships with a global theme stylesheet that must be included in your project.

Update your Angular angular.json styles array:

>_ angular.json

This file:

  • Defines CSS variables
  • Registers background/foreground tokens
  • Enables dark mode support

Tailwind Integration

Tolle UI integrates directly with Tailwind using a preset.

Tailwind Configuration

>_ tailwind.config.js

What This Gives You

  • Access to primary-* color scale
  • Semantic utilities like bg-background
  • Dark mode compatibility
  • Component-aware Tailwind scanning
