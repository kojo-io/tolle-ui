Anatomy

Status maps to the theme's semantic tokens — --success, --warning, --destructive — rather than the chart palette, because a block is a state, not a category.

In a status card

Accessibility

Each block is a real <button>, so it is a tab stop with its own accessible name — the tooltip text if you gave it one, otherwise “Period N”. Hover, keyboard focus and screen readers all reach the same information; colour is never the only channel a block's status is carried on.
