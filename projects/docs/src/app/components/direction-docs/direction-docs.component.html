Installation

Add it with the CLI (you own the source), or import the directive if you use the package.

Examples

Basic

This is a directive on your own element, not a wrapper component — it sets dir on the host and adds no extra DOM.

Switching at runtime

Because the direction is a binding, a locale switch flips the whole subtree without a reload. Prefer logical CSS properties (ms-*, me-*, ps-*) over left/right ones in your own markup so it mirrors with the container.

Reading the direction

Components that need to flip an icon or an animation can read the published value instead of re-deriving it from the DOM.
