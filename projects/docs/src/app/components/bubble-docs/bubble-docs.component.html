Installation

Add it with the CLI (you own the source), or import the components if you use the package.

Examples

Variants and sizes

align squares the bottom corner nearest the sender, so the bubble reads as having a tail. Match it to the enclosing tolle-message.

Actions and reactions

tolle-bubble-actions stays hidden until the bubble is hovered or something inside it takes focus — so a transcript is not a wall of buttons, but the controls are still reachable by keyboard. Clicks are stopped inside the row, so an interactive bubble does not fire as well.

Long answers and interactive bubbles

A long generated answer can bury the rest of the transcript. Set collapsible to clamp it to collapsedHeight behind a toggle. Setting interactive makes the whole surface a focusable trigger for jump-to-thread or open-detail.
