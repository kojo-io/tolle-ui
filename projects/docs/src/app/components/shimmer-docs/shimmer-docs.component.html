Examples

Basic

The projected text sweeps with the .shimmer utility from the theme, so it picks up your foreground and muted tokens.

Sizes

Five text sizes, so the status can sit inline or lead a panel.

Settling to plain text

Set [active]="false" once the work is done and the same label renders plain and static — no unmount, no layout shift. While it is active the label is a role="status" live region.

Accessibility

  • The sweep is a purely decorative motion, so it is disabled under prefers-reduced-motion: reduce — the label falls back to flat muted text rather than a paused, half-transparent gradient.
  • While active the label carries role="status" and aria-live="polite", so screen readers announce the status change without interrupting. Both attributes are dropped once it settles.
  • Keep the label short and meaningful — it is read aloud verbatim each time it changes.
