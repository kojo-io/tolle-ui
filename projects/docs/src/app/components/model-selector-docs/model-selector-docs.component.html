Installation

Add it with the CLI (you own the source), or import the component if you use the package. Bring FormsModule or ReactiveFormsModule along, depending on how you bind it.

Anatomy

One element. It implements ControlValueAccessor and round-trips the chosen model's id, so it binds like any native control. Searching, keyboard navigation and positioning are reused from tolle-command and tolle-popover.

Each entry in models is a ModelOption. Only id and name are required; everything else refines how the option reads.

Examples

With ngModel

The simplest binding: [(ngModel)] holds the selected model's id. Type to filter — the search matches the name, the provider and the description.

Reactive forms and grouping

formControlName works the same way, and the control is marked touched when the list closes. Options carrying a provider are bucketed under a heading in first-appearance order; badge adds a pill and disabled takes an option out of play.

Sizes, variants and disabled

variant="ghost" at size="xs" is the one to reach for inside a composer toolbar; drop the search box with [searchable]="false" for a short list. modelChange hands back the whole ModelOption when the id alone is not enough.
