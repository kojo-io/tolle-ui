Avatar Component Style Override

Avatar Component Style Reference

Component Area Selector What It Targets
Host Element tolle-avatar The component host element with size, shape, and layout styles
Small Size tolle-avatar[size="sm"] 8×8 pixels with extra-small text
Default Size tolle-avatar[size="default"] 10×10 pixels (default size)
Large Size tolle-avatar[size="lg"] 16×16 pixels with large text
Extra Large Size tolle-avatar[size="xl"] 24×24 pixels with extra-large text
Circle Shape tolle-avatar[shape="circle"] Circular avatar with rounded-full
Square Shape tolle-avatar[shape="square"] Square avatar with rounded-md corners
Avatar Image tolle-avatar img Image element when src is provided and loads successfully
Loading Image tolle-avatar img.opacity-0 Image during loading state (hidden)
Fallback Container tolle-avatar > div Container shown when image fails to load or no src provided
Fallback Content tolle-avatar > div > ng-content Projected content (initials, icons, etc.) inside fallback
Muted Background tolle-avatar .bg-muted Background color for error/fallback states
Object Cover tolle-avatar .object-cover Image object-fit: cover styling
Transition Opacity tolle-avatar .transition-opacity Opacity transition for image loading

Avatar Fallback Component Style Reference

Component Area Selector What It Targets
Host Element tolle-avatar-fallback The component host element containing the fallback content
Container Div tolle-avatar-fallback > div Main container div with flex layout and background
Flex Container tolle-avatar-fallback .flex Flex layout utilities for centering content
Background Color tolle-avatar-fallback .bg-muted Background color for fallback state
Text Color tolle-avatar-fallback .text-muted-foreground Text color for fallback content
Font Weight tolle-avatar-fallback .font-medium Medium font weight for fallback text
Text Transform tolle-avatar-fallback .uppercase Uppercase text transformation (for initials)
Projected Content tolle-avatar-fallback ng-content Content projected into the fallback (initials, icons, etc.)
