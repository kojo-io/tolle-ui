Installation

Add it with the CLI (you own the source), or import the component if you use the package.

Anatomy

Only <tolle-table> is a custom element. Everything below it is a directive applied to the native element — <thead tolleTableHeader>, <tr tolleTableRow>, <td tolleTableCell>. There is no <tolle-table-row> element: a wrapper sitting between <tbody> and <tr> would break the CSS table layout, so the directives style the real elements in place and add no DOM of their own.

This is the low-level building block. If you need sorting, paging, selection and column definitions out of the box, reach for tolle-data-table instead.

Examples

Basic

A caption, a header row, and a right-aligned amount column.

With a footer

Rows come from a typed array in the component, and a <tfoot tolleTableFooter> carries the total.

Selectable rows

Bind selected on the row directive to highlight the active record, and set sort on a header cell to publish the current sort direction as aria-sort.
