Basic Usage

The tolle-date-range-picker component is a button trigger — the same shape as tolle-date-time-picker — that opens a range calendar in a floating panel. It supports opening from the keyboard (Enter, Space or an arrow key), a Clear / Done footer, and visual feedback for the selected range.

Two months

Set [numberOfMonths]="2" to render two consecutive months side by side — ideal for booking and reservation flows where the range often spans a month boundary. A single set of prev/next controls moves both months together.
