Modal

A service-driven dialog component that appears on top of the main content.

Overview

Unlike most components in this library, the Modal is managed through the ModalService. This imperative approach allows you to launch dialogs from anywhere in your application logic without needing to manage visibility state in your templates.

Basic Usage (String Content)

Pass a simple string to the content property for text-only messages.

Template Content

Use TemplateRef for richer HTML content defined in your component and pass context data.

Component Content

Inject standalone components for complex logic. Access the modal handle via ModalRef.
