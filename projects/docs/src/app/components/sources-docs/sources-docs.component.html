Installation

Add it with the CLI (you own the source), or import the components if you use the package.

Anatomy

Compose the list from a trigger, its content and one tolle-source per citation. The list owns the shared state and every source registers itself, so the trigger's “Used N sources” label needs no bindings — and is correct before the list has ever been opened.

Examples

Basic

Collapsed by default. Each citation is numbered in DOM order and opens in a new tab; the title falls back to the href when you leave it empty.

Variants and favicons

card and muted give the list a surface of its own. Pass a favicon and it replaces the numbered chip; size="sm" tightens a long list.

Controlled

Bind [(open)] to drive the list from outside — expand it automatically once a response finishes streaming, for instance. openChange reports user toggles only.
