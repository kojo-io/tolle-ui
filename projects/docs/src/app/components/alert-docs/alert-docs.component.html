Alert Component Style Reference

Component Area Selector What It Targets
Alert Container tolle-alert > div Main alert container with border, padding, and animations
Default Variant tolle-alert[variant="default"] > div Default alert styling with neutral colors
Destructive Variant tolle-alert[variant="destructive"] > div Error/alert styling with red/destructive colors
Success Variant tolle-alert[variant="success"] > div Success notification styling with green colors
Warning Variant tolle-alert[variant="warning"] > div Warning notification styling with amber colors
Alert Icon Slot tolle-alert i Alert icon (from <ng-content select="[icon]"> slot)
Alert Title tolle-alert h5 Alert title element (when title input is provided)
Content Wrapper tolle-alert > div > div:last-child Content wrapper div containing title and text
Main Text tolle-alert .text-sm Main text content with smaller font size
Dismiss Button tolle-alert button Close/dismiss button (only when dismissible is true)
Dismiss Icon tolle-alert button i Close icon (×) inside dismiss button
Dismiss Button Hover tolle-alert button:hover Hover state for dismiss button
Dismiss Icon Hover tolle-alert button:hover i Hover state for close icon
Fade Animation tolle-alert .opacity-0 Fade out animation during dismissal
Scale Animation tolle-alert .scale-95 Scale down animation during dismissal
Content Paragraphs tolle-alert p Paragraph elements inside alert content
Group Hover States tolle-alert .group-hover\:opacity-100 Dismiss button opacity on group hover
