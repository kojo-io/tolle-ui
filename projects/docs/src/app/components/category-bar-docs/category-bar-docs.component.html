Installation

Add it with the CLI (you own the source), or import the component if you use the package.

Anatomy

values does not need to sum to 100 — segments are normalized to the bar's own width, the same way tolle-chart-pie normalizes slice values to the circle.

Examples

Basic

Segment labels are your own text underneath — the bar draws no labels itself.

With a marker

markerValue is on the same 0-100 scale the segments sum to — map your own value's range onto it.
