Installation

Add it with the CLI (you own the source), or import the component if you use the package.

Anatomy

A single element. The trigger — a percentage and a small bar — and the hover breakdown are composed for you inside tolle-context, positioned by the existing tolle-hover-card. Any of inputTokens, outputTokens and cost you leave unset simply drops its row from the breakdown.

Examples

Basic

Compact enough to sit in a composer toolbar or a status bar. Hover the indicator for the full breakdown — counts are abbreviated (42K, 1.2M) so a large window still reads at a glance.

Usage levels

The indicator crosses into the warning tokens at warningThreshold (0.75 by default) and the destructive tokens at criticalThreshold (0.9). Both the text and the bar follow the level.

Tracking a session

Feed it the running totals as turns accumulate and the bar, percentage and colour follow. Tighten the thresholds when you want to warn users well before the window is actually full.
