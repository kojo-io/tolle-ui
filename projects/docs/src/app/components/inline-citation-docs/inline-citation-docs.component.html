Installation

Add it with the CLI (you own the source), or import the components if you use the package.

Anatomy

The marker sits directly against the word it cites — no space before the opening tag, or the superscript detaches from the sentence. Whatever you project renders inside the hover card; a tolle-inline-citation-card is the usual choice.

Examples

Basic

Number each claim in reading order. The card shows the title, the URL without its scheme, and a snippet clamped to three lines — hover a marker to see it.

Pulled quote

Project a tolle-inline-citation-quote into the card when the exact wording is the point. It renders as a blockquote with an optional attribution footer.

Variants and sizes

muted and subtle keep markers from speckling a heavily cited answer, and placement moves the card when the marker sits near the top of the viewport.
