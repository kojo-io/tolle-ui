Installation

Add it with the CLI (you own the source), or import the components if you use the package.

Examples

Basic

tolle-message only lays out the row. Put the visible surface inside it with tolle-bubble, and match the bubble's align to the message's.

Grouping a burst of replies

Someone firing off three short lines is one turn, not three. Wrap them in tolle-message-group with tight spacing and repeat the avatar and header once.

Header and footer

Both slots follow the message's alignment automatically — they flip their row direction when align="end", so the timestamp always sits on the outer edge.
