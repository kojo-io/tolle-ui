

# Tolle UI

## Config-first theming for Angular

Tolle UI is designed to be **globally configurable**.
Instead of styling every component individually, you define your design tokens once at the root of your application.

This **config-first** approach allows you to rebrand your entire application in seconds.

---

## 1. Angular Provider

The `provideTolleConfig` function is the brain of the library.
It injects your brand settings into all components using Angularâ€™s **Dependency Injection** system.

### `app.config.ts`

```ts
import { ApplicationConfig, provideZoneChangeDetection } from '@angular/core';
import { provideTolleConfig } from '@tolle_/tolle-ui';

export const appConfig: ApplicationConfig = {
  providers: [
    provideZoneChangeDetection({ eventCoalescing: true }),

    // Define your Tolle UI configuration here
    provideTolleConfig({
      primaryColor: '#551a65', // Your brand color (hex)
      radius: '0.75rem',       // Global border radius
      darkByDefault: false,    // Initial color mode
    }),
  ],
};
```

# Tailwind Integration

Tolle UI components use Tailwind utility classes that reference CSS variables generated by your configuration. To make these work seamlessly, you must integrate the Tolle UI preset into your Tailwind configuration.

## Quick Setup

### 1. Update `tailwind.config.js`

```javascript
/** @type {import('tailwindcss').Config} */
module.exports = {
  // Load the Tolle UI preset first
  presets: [
    require('@tolle_/tolle-ui/preset')
  ],

  // Ensure Tailwind scans your app and the library
  content: [
    // Your application files
    "./src/**/*.{html,ts}",
    "./projects/**/*.{html,ts}",

    // Essential: Scan the Tolle UI library for class usage
    "./node_modules/@tolle_/tolle-ui/**/*.{html,ts,mjs}"
  ],

  theme: {
    extend: {
      // You can extend the theme here with custom values
      // These will override or extend the Tolle UI preset
    },
  },

  plugins: [],
}
```

# Global Styles

Add the base theme file to your angular.json (or project.json). This file contains the foundational CSS resets required for the components.

```css
"styles": [
  "src/styles.css",
  "node_modules/@tolle_/tolle-ui/theme.css"
]
```

# Theme Variables

Tolle UI generates a comprehensive system of CSS custom properties (CSS variables) based on your configuration. These variables ensure consistent theming across all components and enable easy customization.

## Core Configuration Variables

These are the variables you define in your `provideTolleConfig()` function:


| Variable    | Type  | Default   | Description                        |
|------------|-------|-----------|------------------------------------|
| `--primary` | Color | `#2563eb` | The primary brand color (blue-600) |
| `--radius`  | Size  | `0.5rem`  | Global border radius for components |


### Light Mode Palette
```css
:root {
  --primary-50: #eff6ff;    /* Lightest blue */
  --primary-100: #dbeafe;   /* Very light blue */
  --primary-200: #bfdbfe;   /* Light blue */
  --primary-300: #93c5fd;   /* Medium-light blue */
  --primary-400: #60a5fa;   /* Blue */
  --primary-500: #3b82f6;   /* Primary blue */
  --primary-600: #2563eb;   /* Your configured primary */
  --primary-700: #1d4ed8;   /* Dark blue */
  --primary-800: #1e40af;   /* Darker blue */
  --primary-900: #1e3a8a;   /* Darkest blue */
}
```
